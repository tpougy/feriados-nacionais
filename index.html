Feriados Nacionais da ANBIMA

Este repositório contém os dados dos feriados nacionais brasileiros, extraídos da ANBIMA e disponibilizados em diversos formatos. Os dados são atualizados anualmente via GitHub Actions.

Documentação disponível em https://github.com/tpougy/feriados-nacionais
